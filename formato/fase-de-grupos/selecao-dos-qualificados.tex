\subsubsection{Seleção dos Qualificados}

Para a seleção dos 8 times qualificados a prosseguir para a próxima fase, serão usados os seguintes critérios, em ordem de prioridade:

\vbox{%
    \begin{itemize}
        \item igual distribuição da quantidade de times qualificados por grupo;
        \item colocação dentro de cada grupo.
    \end{itemize}
}

Nos casos em que os critérios acima não forem suficientes para determinação dos classificados para a próxima fase, fica a cargo da organização a definição de novos critérios, podendo esta inclusive fazer leves alterações no cronograma para comportar partidas de desempate.
