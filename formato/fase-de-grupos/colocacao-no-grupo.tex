\subsubsection{Colocação no Grupo}

Para a definição da colocação de cada um dos times dentro de cada grupo, serão usados os seguintes critérios, em ordem de prioridade:

\vbox{%
	\begin{enumerate}
		\item maior número de vitórias;
		\item confrontos diretos;
		\item
			\begin{enumerate}
				\item no LoL, menor média do tempo de vitórias;
				\item no CS:GO, maior número de rounds vitoriosos;
			\end{enumerate}
		\item
			\begin{enumerate}
				\item no LoL, maior média do ouro total do time;
				\item no CS:GO, maior número de eliminações;
			\end{enumerate}
	\end{enumerate}
}

Nos casos em que os critérios acima não forem suficientes para determinação das colocações necessárias para definir os classificados para a próxima fase, fica a cargo da organização a definição de novos critérios, podendo esta inclusive fazer leves alterações no cronograma para comportar partidas de desempate.
